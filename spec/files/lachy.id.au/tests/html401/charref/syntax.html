Character Reference Syntax Tests

The following tests browser support for the syntax of hexadecimal and numeric character references and entity references with and without the REFC (Reference Close) delimiter (;) for the character references defined in the HTML 4.01 Recommendation. Note that &apos; is not defined in HTML 4.01, but it is included because some browsers do support it.
